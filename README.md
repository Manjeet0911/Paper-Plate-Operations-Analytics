Paper Plate Manufacturing Analytics Dashboard
Project Overview
An end-to-end data intelligence project to track sales and material consumption for a manufacturing unit.

Tech Stack Used
Python: Data generation (500+ records).

MySQL: Database management and structured querying.

Power BI: Interactive dashboarding and business insights.

Key Insights
Top Seller: 13-inch plates generated the highest revenue ($0.4M).

Seasonal Trend: Significant spike in raw material usage during March.

Market Share: Balanced distribution across all product sizes (6", 7", 12", 13").
